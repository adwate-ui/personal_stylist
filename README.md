## Supabase Setup

1. Run migration: Paste the contents of `supabase/migrations/001_unified_schema.sql` into the Supabase SQL Editor and run it.
2. Verify buckets exist in Storage dashboard:
   - `wardrobe_items` (public, 5MB limit)
   - `user_uploads` (public, 5MB limit)
   - `avatars` (private, 2MB limit)
3. Ensure RLS is enabled on all tables (`wardrobe_items`, `profiles`, `shopping_items`).
4. Test policies with a test user account.

### Environment Variables
Ensure your `.env.local` or Cloudflare Pages environment variables are set:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY` (for admin operations)
